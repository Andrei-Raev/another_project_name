В корневом каталоге находится 3 папки. 
1. res: в ней храняться ресурсы игры. Пока что там есть единственная папка "fonts", в которой хранится шрифт для игры
2. sample_classes: хранит образцы классов и наработки по ним, что бы потом быть интегрированы в игру (к примеру там есть антиалиасинг или экран загрузки)
3. work_dir: хранит наработки по самой игре. Содержит 3 папки:
3.1. Main_menu: главное меню игры (требуется доработка)
3.2. map_gen: генератор карты шумом перлина
3.3. world_class: содержит текущую версию игры

Рабочий билд актуальной версии находится в папке "bild" в корневом каталоге
